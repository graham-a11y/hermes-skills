# Hermes Skills

Custom skills for Hermes Agent.

## Skills

### fireflies-to-tweets
Turn Fireflies meeting transcripts into audience-resonant tweets and threads using the 4-System AI Content Engine. Extracts signal through both brand strategy and reader empathy — not just "on brand," but "would a stranger feel something reading this?"

**Install:**
```bash
hermes skills tap add graham-a11y/hermes-skills
hermes skills install fireflies-to-tweets
```

**Update (when new versions drop):**
```bash
hermes skills check && hermes skills update
```

### hermes-tweet
Search tweets, read replies, look up users, monitor trends, post tweets, send DMs, and gate X/Twitter account actions through Xquik from Hermes Agent.

**Install:**
```bash
hermes skills tap add graham-a11y/hermes-skills
hermes skills install hermes-tweet
```
